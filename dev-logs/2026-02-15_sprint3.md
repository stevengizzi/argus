# Sprint 3 Implementation — Strategy Framework + ORB Breakout

**Date:** 2026-02-15
**Sprint:** 3
**Tests:** 222 passing

## Summary

Implemented the complete strategy framework and first trading strategy (Opening Range Breakout). This sprint establishes the patterns all future strategies will follow.

## Components Implemented

### Core Updates
- **DEC-037 Fix**: Risk Manager now uses `_start_of_day_equity` for cash reserve calculations, snapshotted in `reset_daily_state()`
- **Deprecated API Fix**: Replaced all `datetime.utcnow()` with `datetime.now(UTC)` (DEF-003)

### New Config Models (`argus/core/config.py`)
- `DataServiceConfig` — timeframes, indicators, stale data timeout
- `ScannerConfig` — scanner type and static symbols
- `OrbBreakoutConfig` — ORB-specific parameters (window, chase protection, volume multiplier)

### New Strategy Models (`argus/models/strategy.py`)
- `ScannerCriteria` — min/max price, volume, gap filters
- `ProfitTarget` — target level with partial exit percentage
- `ExitRules` — profit targets, time stop, trail activation
- `MarketConditionsFilter` — VIX, SPY slope, RVOL filters
- `StrategyWatchlistItem` — symbol with strategy-specific metadata

### Abstractions
- `BaseStrategy` (`argus/strategies/base_strategy.py`) — ABC defining strategy interface with daily state reset, position sizing, exit rules, market condition filtering
- `Scanner` (`argus/data/scanner.py`) — ABC for stock selection
- `StaticScanner` — returns configured symbol list (for replay/backtest)
- `DataService` (`argus/data/service.py`) — ABC for market data access

### ReplayDataService (`argus/data/replay_data_service.py`)
- Reads Parquet files for historical data replay
- Publishes `CandleEvent` and `IndicatorEvent` to EventBus
- Computes indicators: VWAP (daily reset), ATR(14) with Wilder's smoothing, SMA(9/20/50), RVOL
- Configurable replay speed (0 = instant, 1.0 = real-time)

### OrbBreakoutStrategy (`argus/strategies/orb_breakout.py`)
- Tracks opening range (OR) during configurable window (default 15 min)
- Validates OR against ATR bounds (0.5x - 3.0x ATR)
- Detects breakouts: close > OR high, volume > threshold, price > VWAP
- Chase protection: rejects entries too far above OR high
- Position sizing: risk dollars / risk per share
- Stop placement: OR midpoint
